# 岩心孔洞裂缝分析系统 — 设计文档

> **日期**: 2026-06-22
> **版本**: v1.0
> **架构**: Flask Web 应用 + 浏览器桌面化

---

## 1. 系统概述

Flask Web 应用，双击启动自动打开浏览器。内核复用旧项目 7 个引擎模块（57 tests），孔洞/裂缝算法融合学长项目。

## 2. 技术选型

| 层 | 技术 | 理由 |
|---|---|---|
| 后端 | Flask + Jinja2 | 轻量，无 PySide6 依赖 |
| 前端 | HTML5 + Chart.js | 浏览器天然完美交互 |
| 引擎 | OpenCV + NumPy + SciPy | 直接搬旧项目，57 tests |
| 数据库 | SQLite | 单文件，零配置 |
| 打包 | PyInstaller + 自动开浏览器 | ~60MB exe |

## 4. 核心功能

### 4.1 孔洞分析（学长算法 + 旧项目分类）

```
灰度化 → GaussianBlur → 固定阈值(THRESH_BINARY_INV) → 开闭运算 → 轮廓提取
→ 面积过滤(min/max) → 等效直径Dr/面孔率/大小分类/充填统计 → 报告
```

参数：阈值(50-200)、最小面积(px)、最大面积(px)

### 4.2 裂缝分析（学长距离变换 + 旧项目成因评价）

```
灰度化 → CLAHE → BilateralFilter → 自适应+全局阈值 → 形态学 → 区域生长去噪
→ 轮廓提取 → 实体度过滤(<0.7) → 距离变换测宽 → 方向判断(横向/纵向)
→ 成因分类(构造/成岩/风化) → 有效性评价
```

参数：最小面积、阈值、实体度

### 4.3 粒度分析（旧项目 Feret 分析）

```
颜色分割(LAB) → 轮廓提取 → Feret直径/圆度 → Udden-Wentworth分类
→ 频率直方图/累计曲线
```

参数：颜色匹配度、去噪阈值

### 4.4 报告生成
- HTML 报告（Jinja2 模板 + matplotlib 图表 base64 内嵌）
- JSON/CSV 数据导出
- 结果图可保存

### 4.5 知识库
- 沉积学知识 JSON（6 分类 34 条目）
- 搜索 + 超链接跳转

## 5. UI 设计

四视图切换（原图/灰度图/二值图/结果标记图）+ 右侧参数面板 + 底部图表区域

菜单：打开图像 | 孔洞分析 | 裂缝分析 | 粒度分析 | 知识库 | 报告

不包含手动画笔/橡皮擦/膨胀腐蚀——通过参数调节自动完成。
